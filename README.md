## 🎯 What Is This?

A curated collection of **15 AI SaaS landing page projects**, each with:

- 💡 A catchy, marketable product name
- 📝 A compelling short description
- 🎯 A clearly defined target audience
- 🖥️ A **full, copy-paste-ready Framer AI Wireframer prompt** to generate a premium dark-mode landing page instantly

Each prompt generates a page with: **Hero, Problem, How It Works, Features, Testimonials, Pricing, FAQ, and CTA** sections — all styled in dark futuristic mode with neon accents, glassmorphism, and scroll animations.

## 📖 How to Use

1. **Open [PROJECTS.md](PROJECTS.md)** to see all 15 projects with full details
2. **Pick a project** that matches a pain point you want to target
3. **Copy the Framer AI prompt** directly from the project listing
4. **Paste into [Framer AI Wireframer](https://www.framer.com)** and generate your landing page
5. **Customize** colors, copy, and images to match your brand
6. **Ship it** 🚀

## 🎨 Visual Style (All Prompts)

Every prompt generates landing pages with these design specs:

- 🌑 **Dark mode** background (`#0A0A0F`)
- 💎 **Neon cyan** (`#00F0FF`) + **purple/magenta** (`#B026FF`) accents
- 🪟 **Glassmorphism** cards with frosted-glass borders
- 📐 **Bento grid** layout for features
- ✨ **Scroll-triggered animations** (fade-up, parallax, staggered reveals)
- 🖱️ **Smooth hover effects** on all interactive elements
- 🔤 **Modern sans-serif typography** (Inter or Satoshi)
- 🌐 **Bilingual-ready** (English primary)

Inspired by: **Lovable**, **Cursor**, **Arc** aesthetics.
