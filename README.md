## 🎯 What Is This?

A curated collection of **30 AI SaaS landing page projects** across 2 volumes, each with:

- 💡 A catchy, marketable product name
- 📝 A compelling short description
- 🎯 A clearly defined target audience
- 🖥️ A **full, copy-paste-ready Framer AI Wireframer prompt**

Each prompt generates a premium dark-mode landing page with: **Hero, Problem, How It Works, Features, Testimonials, Pricing, FAQ, and CTA** sections.

## 🎨 Visual Style (All Prompts)

Every prompt generates landing pages with these design specs:

- 🌑 **Dark mode** background (`#0A0A0F`)
- 💎 **Neon cyan** (`#00F0FF`) + **purple/magenta** (`#B026FF`) accents
- 🪟 **Glassmorphism** cards with frosted-glass borders
- 📐 **Bento grid** layout for features
- ✨ **Scroll-triggered animations** (fade-up, parallax, staggered reveals)
- 🖱️ **Smooth hover effects** on all interactive elements
- 🔤 **Modern sans-serif typography** (Inter or Satoshi)
- 🌐 **Bilingual-ready** (English primary)

Inspired by: **Lovable**, **Cursor**, **Arc** aesthetics.

## 📖 How to Use

1. **Pick a volume:** [V1](PROJECTS.md) (concise) or [V2](PROJECTS_V2.md) (expanded with features & pricing)
2. **Choose a project** that matches a pain point you want to target
3. **Copy the Framer AI prompt** from the project listing
4. **Paste into [Framer AI Wireframer](https://www.framer.com)** and generate
5. **Customize** and **ship** 🚀
